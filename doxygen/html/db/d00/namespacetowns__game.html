◆ load_towns() [1/2]

void towns_game::load_towns ( const char * all_towns_txt)

Возвращает название города, содержащееся в all_towns и не содержащщеся в logs, начинающееся на последнюю букву последнего элемента logs.

Эта функция перегружена и предоставляется исключительно для удобства использования. Она отличается от вышеупомянутой только фактическими аргументами.

◆ load_towns() [2/2]

void towns_game::load_towns ( const std::string & all_files_txt)

Загружает названия городов из текстового файла в towns_game::all_towns.

Формат входного файла: все названия, разделенные переходом на следующую строку. Все символы в названиях, не являющиеся буквами, заменяются на пробелы в all_towns и, таким образом, равносильны для программы

Аргументы
[in]destНазвание файла с именами городов
Исключения
std::runtime_errorЕсли указанный файл не найден
